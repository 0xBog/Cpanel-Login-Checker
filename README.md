# Cpanel Login Checker

## Installation

Install Cpanel Login Checker


```bash
 Python 3.x
```

```bash
  git clone https://github.com/0xBog/Cpanel-Login-Checker

  cd Cpanel-Login-Checker-main

  pip install -r requirements.txt

  python Cpanel-Checker.py
```
